DeliCube rules:


-No Hacking or Advantageous Modifications: Any use of hacks or mods that give unfair advantages is strictly prohibited.

-Exploitation: Exploiting game mechanics or bugs for personal gain or disrupting server play is not allowed.

-Lag-causing Actions: Creating lag intentionally or using machines that cause server performance issues is prohibited.

-Bug Reporting: Always report bugs through proper channels. Exploiting unreported bugs for personal gain is forbidden.

-Impersonation: Do not impersonate other players or moderators on the network. Being genuine and using one's own identity is expected.


Remember, following these rules helps create a fair and enjoyable gaming environment for everyone on the DeliCube Network!
